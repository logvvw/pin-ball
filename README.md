# 乒乓球大师

奥运会风格的乒乓球对打游戏，玩家与AI对战，5局3胜晋级制度。

## 游戏特性

- 🏓 **真实乒乓球玩法**: 玩家与AI进行乒乓球对打
- 🎮 **滑动控制**: 手指左右滑动控制球拍移动
- 🏆 **奥运会赛制**: 5局3胜淘汰赛，赢取晋级
- 🤖 **5级AI难度**: 从业余选手到奥运冠军，逐级挑战
- 📈 **难度递增**: 击败当前对手，进入更高难度
- ✨ **粒子效果**: 得分时显示炫酷动画
- 🎯 **真实物理**: 球的旋转、角度、速度模拟真实乒乓球

## 游戏规则

1. **基础规则**: 11分制，领先2分获胜
2. **发球规则**: 每方连续发2球后换发，平分后每次换发
3. **赛制**: 5局3胜，赢得比赛进入下一轮
4. **晋级路线**: 热身赛 → 16强 → 8强 → 4强 → 决赛

## 操作方法

- **滑动控制**: 在屏幕上左右滑动控制球拍
- **点击**: 在开始/结束界面点击按钮

## 赛事阶段

| 阶段 | AI难度 | 说明 |
|------|--------|------|
| 热身赛 | 业余选手 | 1局1胜 |
| 16强 | 业余高手 | 5局3胜 |
| 8强 | 半职业选手 | 5局3胜 |
| 4强 | 职业选手 | 5局3胜 |
| 决赛 | 奥运冠军 | 5局3胜 |

## 技术特点

- 使用 Canvas 2D 进行渲染
- 模块化设计（球拍、球、比赛、AI独立类）
- requestAnimationFrame 游戏循环
- 帧率独立的物理计算
- 响应式设计，适配不同屏幕尺寸

## 开发说明

这是从弹球游戏重构为乒乓球对战的微信小游戏项目。

主要架构：
1. **Ball.js**: 乒乓球运动和碰撞逻辑
2. **Paddle.js**: 球拍控制和渲染
3. **Table.js**: 球台和球网绘制
4. **Match.js**: 比赛计分和规则
5. **AI.js**: 5级难度AI对手
6. **UI.js**: 比分面板和游戏界面
