# 弹球小游戏

这是一个微信小游戏版本的弹球游戏，小球会在屏幕边缘自动反弹。

## 游戏特性

- 🎮 **自动反弹**: 小球在屏幕边缘自动反弹
- 🎯 **触摸生成**: 触摸屏幕任意位置生成新的小球
- ✨ **粒子效果**: 碰撞时产生绚丽的粒子爆炸效果
- 🎨 **拖尾效果**: 小球移动时带有炫酷的拖尾
- 📈 **连击系统**: 连续碰撞获得额外加分
- 🏆 **难度递增**: 随着分数增加，游戏难度逐渐提升

## 游戏规则

1. 游戏自动开始，屏幕上会生成3个初始小球
2. 小球碰到屏幕边缘会反弹并得分
3. 连续碰撞会获得连击加分
4. 触摸屏幕可以手动生成高分小球
5. 随着等级提升，小球生成速度加快，最大数量增加

## 操作方法

- **触摸屏幕**: 在触摸位置生成新的小球
- **屏幕边缘反弹**: 小球自动在屏幕边缘反弹

## 技术特点

- 使用 Canvas 2D 进行渲染
- 物理碰撞检测和反弹计算
- 粒子系统特效
- 响应式设计，适配不同屏幕尺寸

## 开发说明

这是一个从微信小程序转换为微信小游戏的示例项目。主要的改动包括：

1. 移除小程序的页面结构（WXML、WXSS等）
2. 使用 Canvas API 替代小程序的画布组件
3. 适配小游戏的触摸事件系统
4. 重构游戏循环以适应小游戏环境
